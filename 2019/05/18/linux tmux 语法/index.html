linux tmux 语法

tmux 参数

  • new -s djangoBlog 创建会话
  • a -t djangoBlog进入会话
  • ls会话列表

按下Ctrl+B后

修改窗口

  • % 左右平分出两个窗格
  • " 上下平分出两个窗格
  • { 当前窗格前移
  • } 当前窗格后移
  • 同时按方向键调整窗口大小

操作窗口

  • 方向键切换窗口
  • t 窗口显示时间
  • $ 重命名
  • b回到bash
  • k结束会话
  • x 关闭当前窗格
